## [1.0.0] - 2025-01-15

### Added
- Production profile `README.md` centered on "Hassan Mubiru — Creator of StreetJS" with hero, founder identity, tech stack, StreetJS section, featured projects, open source, GitHub stats, streak, trophies, activity graph, and contact sections.
- Brand banner (`assets/banner.png`) and Open Graph image (`assets/og-image.png`) using the six-color brand palette.
- Community health files: `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, and `SECURITY.md`.
- GitHub issue templates (`.github/ISSUE_TEMPLATE/`) for bug reports, feature requests, and documentation issues, plus a configuration that routes questions to Discussions.
- Pull request template (`.github/pull_request_template.md`).
- `pinned-repositories.md` recommending the six pinned repositories and their order.
- `repository-metadata.md` with descriptions and topics for StreetJS, Vibra, Afrchat, and SCMINER.
- Repository health files: `LICENSE` (MIT), `CHANGELOG.md`, and `ROADMAP.md`.

### Changed
- Standardized all brand color references to the palette: `#2563EB`, `#06B6D4`, `#0F172A`, `#111827`, `#8B5CF6`, `#F8FAFC`.
